**৮. কাস্টমারঃ** জ্বী, দেখুন। আর একটা কথা, শুধু speed কম না, মাঝে মাঝে connection drop-ও হচ্ছে। ৫-১০ মিনিট পরপর disconnect হয়ে আবার connect হয়।

**৯. এজেন্টঃ** sir, diagnostics result দেখে বলছি — আপনার ONU device-এর optical signal level টা normal range-এর নিচে আছে। স্বাভাবিক হওয়া উচিত মাইনাস ১৫ থেকে মাইনাস ২৫ dBm-এর মধ্যে, কিন্তু আপনার টা দেখাচ্ছে মাইনাস ২৯ dBm। এর মানে হলো fiber cable-এ কোথাও signal loss হচ্ছে, সম্ভবত cable-এর কোনো joint-এ বা bend-এ সমস্যা আছে।

**১০. কাস্টমারঃ** তার মানে router-এর সমস্যা না? Cable-এর সমস্যা? এটা ঠিক করতে কি technician আসতে হবে?

**১১. এজেন্টঃ** sir, হ্যাঁ, এটা router-এর সমস্যা না। Fiber cable-এর কোনো point-এ signal degradation হচ্ছে, এটা fix করতে আমাদের technician-কে physically এসে cable check করতে হবে। আমি এখনই একটা technician visit schedule করে দিচ্ছি।

**১২. কাস্টমারঃ** কবে আসবে technician? আমি বললাম তো, আজকে রাতের মধ্যে file upload করতে হবে, নাহলে client-এর deadline miss হবে।

**১৩. এজেন্টঃ** sir, আপনার urgency বুঝতে পারছি। আমি urgent visit request দিচ্ছি, আজকেই বিকেলের মধ্যে technician পাঠানোর চেষ্টা করব। তবে এর মাঝে আমি আমাদের end থেকে একটা কাজ করতে পারি — আপনার port-টা আমি reset করে দিচ্ছি, এতে temporarily speed কিছুটা improve হতে পারে।

**১৪. কাস্টমারঃ** সেটা করুন। আর technician-এর ব্যাপারটা confirm করেন please। আমার সারাদিন বাসায় আছি, যেকোনো সময় আসতে পারে।

**১৫. এজেন্টঃ** sir, আমি port reset করে দিলাম। আপনার router-টা একবার power off করে ৩০ সেকেন্ড wait করে আবার on করুন। তারপর ২-৩ মিনিট পর speed test করে দেখুন কোনো improvement আসে কিনা।

**১৬. কাস্টমারঃ** ঠিক আছে, করছি। ধরুন। ... হ্যাঁ, router on হয়েছে। ... Speed test করছি। ... ১৮ Mbps দেখাচ্ছে এখন। আগের চেয়ে ভালো, কিন্তু ৪০ Mbps তো না।

**১৭. এজেন্টঃ** sir, হ্যাঁ, port reset-এ কিছুটা improve হয়েছে, তবে full speed পেতে হলে cable-এর physical issue টা fix করতেই হবে। আমি technician visit আজকে বিকেল ৪টা থেকে ৬টার মধ্যে schedule করেছি, ticket number হলো LNK-SPEED-২০২৫-০১২৩। Technician আসার আগে আপনাকে call করবে।

**১৮. কাস্টমারঃ** ঠিক আছে, বিকেলে আসুক। আচ্ছা একটা কথা, এই যে তিনদিন আমি কম speed পেলাম, এটার কি কোনো refund বা adjustment হবে?

**১৯. এজেন্টঃ** sir, আপনার concern valid। আমি আপনার ticket-এ note করে দিচ্ছি যে গত তিনদিন speed issue ছিল। Technician visit-এর পরে issue resolve হলে, আপনি আমাদের billing team-এ call করে এই ticket number reference দিয়ে bill adjustment request করতে পারবেন।

**২০. কাস্টমারঃ** মানে আবার আমাকেই call করতে হবে? আপনারা automatically adjust করতে পারেন না?
